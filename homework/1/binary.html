The binary search is a method by which we can search through a sorted array (or Binary Search Tree (BST))

It functions by taking a target value, and comparing it to the middle-most value of the data structure (the middle index of an array, or the root of a BST

In the case that the value is higher than the middle, it repeats on the middle-most value of the higher half, and the same for the lower half if the value was lower.

This process is repeated until the target value is found, or the array cannot be divided any further (or the tree has no more child nodes)

In this example, the target value is 7.

The algorithm first finds the middle-most value, in this case 14.

Since 7 < 14, the algorithm finds the middle-most value to the left, 6.

The comparison is made again, this time, 7 > 6, so the middle-most value to the right is selected.

We're getting close now! Our value is now 8; we compare (7 < 8), and find the middle-most value to the left.

After checking this value agains our target, we know we've found it (7 = 7)!

Requirements

Binary search requires either random access, or a binary search tree

Binary search only works on sorted data
