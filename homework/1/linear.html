Linear search is the simplest of the algorithms,

It functions by iterating one element at a time through the data structure until it finds an element equaling it's target

Here we see the algorithm iterating through the values one by one until it finds it's target J; at which point it drops out.

Requirements

Linear search can be used on any array or linear data structure, even if it's unsorted.

Linear search is considered the most basic search, and contains no optimisations.
